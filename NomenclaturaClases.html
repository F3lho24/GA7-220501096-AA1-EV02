Nomenclatura: Clases, Métodos y Comentarios

Convenciones para escribir código limpio y profesional

Clases

Java / C#

// PascalCase + sustantivos
public class UserService {
    // Campos
    private int maxAttempts;

    // Constructor
    public UserService() {...}
}
  • Usar PascalCase (PrimeraLetraMayúscula)
  • Nombres deben ser sustantivos
  • Evitar prefijos como "C_" o "cls"

Python

# PascalCase + sin guiones
class DatabaseConnection:
    def __init__(self):
        self._max_retries = 3  # Variable interna
  • PascalCase aunque el lenguaje acepte snake_case
  • Para variables internas usar _prefix

Métodos / Funciones

JavaScript

// camelCase + verbo
function calculateTotal(items) {
    return items.reduce((sum, item) => sum + item.price, 0);
}

// Método privado
function _validateInput(input) {...}
  • Usar camelCase
  • Comenzar con verbo (get, calculate, validate)
  • Métodos privados con _prefix

Python

# snake_case + verbo
def calculate_tax(amount):
    return amount * 0.19

# Método privado
def _internal_helper():
  • Usar snake_case
  • Verbo + contexto (ej: get_user_by_id)

Comentarios

Documentación

/**
 * Calcula el impuesto para un producto
 * @param {number} price - Precio base
 * @param {string} category - Categoría fiscal
 * @returns {number} Impuesto calculado
 */
function calculateTax(price, category) { ... }
  • Usar JSDoc (JavaScript), JavaDoc o docstrings (Python)
  • Documentar parámetros, retorno y propósito

Comentarios de Línea

# MAL: Explica QUÉ hace (redundante)
x = x + 1  # Incrementa x en 1

# BIEN: Explica POR QUÉ
x = x + 1  # Ajuste por desfase horario UTC
  • Evitar comentarios obvios
  • Explicar "por qué" no "qué"
  • Máximo 1 línea cada 10 de código

Anti-patrones Comunes

Nombres genéricos

class Handler, function process()

Solución: PaymentProcessor, validateUserCredentials()

Notación inconsistente

class user_manager (Python con snake_case)

Solución: class UserManager (PascalCase)

Comentarios obsoletos

Código modificado sin actualizar comentarios

Solución: Eliminar o actualizar con cada cambio
